## Claude デスクトップアプリって何？

Claude は普段ブラウザ（claude.ai）からでも使えますが、Anthropic が公式で出している **デスクトップアプリ** を入れておくと、日常の使い勝手がぐっと良くなります。

- ブラウザのタブを開かなくてもアプリアイコンから一発起動
- ショートカットキーで即座に呼び出せる
- ファイル添付や画像の共有がドラッグ & ドロップでスッと済む
- Mac / Windows どちらでも無料で使える

AI をちょっと試したい人から、毎日使い倒したい人まで、**まずはデスクトップ版から入る** のがおすすめです。

## 入手は必ず公式サイトから

インストーラは Anthropic 公式のサポートページからダウンロードできます。

📘 **[Installing Claude for Desktop（Anthropic 公式サポート）](https://support.anthropic.com/en/articles/10065433-installing-claude-for-desktop)**

リンク先から、自分の OS 用のインストーラを落とせば OK。**非公式サイトからのダウンロードは避ける** のが鉄則です（偽物インストーラでの被害が少なくありません）。

### デスクトップアプリで使える3つの機能

インストール後に覚えておきたいのが、デスクトップアプリには **「チャット」「Cowork」「Claude Code」** という3つの使い方があるという点です。

- **チャット** — ブラウザ版と同じ感覚で使える一番基本の機能
- **Cowork** — デスクトップ上のファイルやアプリを横断して動いてくれるエージェント機能
- **Claude Code** — コーディングに特化した CLI ベースの相棒（詳細は次回の記事で）

まず **チャット機能から入って、慣れてきたら Cowork と Claude Code を追加で触る** のが王道ルートです。

## インストール後にやっておきたい3つの設定

1. **アカウントでサインイン** — ブラウザ版と同じ Anthropic アカウントで OK
2. **スタートアップ登録** — PC 起動時に自動起動（Windows はタスクマネージャー、Mac はシステム設定）
