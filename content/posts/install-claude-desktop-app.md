## Claude デスクトップアプリって何？

Claude は普段ブラウザ（claude.ai）からでも使えますが、Anthropic が公式で出しているデスクトップアプリを入れておくと、日常の使い勝手がぐっと変わります。

- ブラウザのタブを開かなくてもアプリアイコンから一発起動
- ショートカットキーで即座に呼び出せる
- ファイル添付や画像の共有がドラッグ & ドロップでスッと済む
- Mac / Windows どちらでも無料で使える

AI をちょっと試したい人から毎日使い倒したい人まで、まずはデスクトップ版から入るのがおすすめです。

## 入手は必ず公式サイトから

インストーラは Anthropic 公式のサポートページから落とせます。

📘 **[Installing Claude for Desktop（Anthropic 公式サポート）](https://support.anthropic.com/en/articles/10065433-installing-claude-for-desktop)**

リンク先から自分の OS 用のインストーラをダウンロードすれば OK。非公式サイトからのダウンロードは避けるのが鉄則です（偽物インストーラでの被害が少なくないので）。

### デスクトップアプリで使える3つの機能

インストール後に覚えておきたいのが、デスクトップアプリには「チャット」「Cowork」「Claude Code」という3つの使い方があるという点です。

- **チャット**: ブラウザ版と同じ感覚で使える一番基本の機能（Free プランでも使える）
- **Cowork**: デスクトップ上のファイルやアプリを横断して動いてくれるエージェント機能（**有料プラン**）
- **Claude Code**: コーディングに特化した CLI ベースの相棒（**有料プラン**／詳細は次回の記事で）

まずチャット機能から入って、慣れてきたら Cowork と Claude Code を追加で触るのが王道ルートです。

## Cowork と Claude Code は有料プランの機能

ひとつだけ先に伝えておきたい大事なこと。**Cowork と Claude Code は、基本的に有料プランの機能** です。

無料プランでもチャット機能は普通に使えますが、この2つを触りたい場合は Pro 以上のプランに加入する必要があります。「デスクトップアプリ自体は無料でダウンロードできるけど、中で使える機能はプランによって変わる」という理解で OK。

### プラン別比較表

2026 年 4 月時点の、主なプランと使用量の目安をまとめます。使用量は公式が出している「1セッションあたり Pro の何倍か」という基準で並べています（Pro = 1倍）。

| プラン | 月額（USD）| チャット | Cowork | Claude Code | 使用量（Pro 基準）|
|---|---|:---:|:---:|:---:|---|
| **Free** | $0 | ○ | ✗ | ✗ | 約 1/5 |
| **Pro** | $17〜$20 | ○ | ○ | ○ | 1倍（基準）|
| **Max 5x** | $100 | ○ | ○ | ○ | 5倍 |
| **Max 20x** | $200 | ○ | ○ | ○ | 20倍 |
| **Team（Standard）** | $20〜$25 / 人 | ○ | ○ | ○ | Pro より多い（詳細非公開）|
| **Team（Premium）** | $100〜$125 / 人 | ○ | ○ | ○ | Standard の 5倍 |
| **Enterprise** | 応相談 | ○ | ○ | ○ | 個別設計 |

※ 料金は変動します。最新は [Anthropic 公式の料金ページ](https://claude.com/pricing) でご確認ください。年払いだと月額が少し安くなります。<br>
※ 倍率は Anthropic 公式の「1セッションあたりの使用量」に基づきます。日毎・月毎の具体的なメッセージ数・トークン数の上限は公式には開示されていません。

### どれを選べばいい？

私自身が最初の頃に悩んだポイントも含めて、選び方の指針です。

- **とりあえず触ってみたい** → **Free** でチャットから試す
- **個人で Cowork / Claude Code を使いたい** → **Pro（月 $17〜$20）** がスタートライン。コスパ最強
- **開発で Claude Code をそこそこ使いたい** → **Max 5x（$100）**。Pro だと頻繁に上限に当たる人はここ
- **本気で Claude Code をブン回す / 一日中 AI と作業する** → **Max 20x（$200）**。使用量の上限をほぼ気にしなくてよくなる
- **チームで共有したい** → **Team（1人あたり $20〜）** で管理機能付き

私のおすすめは **Pro → 物足りなくなったら Max 5x → さらに足りなければ Max 20x** というステップ。いきなり Max 20x にしてもリターンを活かし切れないことが多いので、まずは Pro でしばらく使い倒してみるのが遠回りに見えて近道です。
