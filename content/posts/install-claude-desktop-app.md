## Claude デスクトップアプリって何？

Claude は普段ブラウザ（claude.ai）からでも使えますが、Anthropic が公式で出している **デスクトップアプリ** を入れておくと、日常の使い勝手がぐっと良くなります。

- ブラウザのタブを開かなくてもアプリアイコンから一発起動
- ショートカットキーで即座に呼び出せる
- ファイル添付や画像の共有がドラッグ & ドロップでスッと済む
- Mac / Windows どちらでも無料で使える

AI をちょっと試したい人から、毎日使い倒したい人まで、**まずはデスクトップ版から入る** のがおすすめです。

## 入手は必ず公式サイトから

### デスクトップアプリで使える3つの機能

インストール後に覚えておきたいのが、デスクトップアプリには **「チャット」「Cowork」「Claude Code」** という3つの使い方があるという点です。

- **チャット** — ブラウザ版と同じ感覚で使える一番基本の機能（Free プランでも使える）
- **Cowork** — デスクトップ上のファイルやアプリを横断して動いてくれるエージェント機能（**有料プラン**）
- **Claude Code** — コーディングに特化した CLI ベースの相棒（**有料プラン**／詳細は次回の記事で）

まず **チャット機能から入って、慣れてきたら Cowork と Claude Code を追加で触る** のが王道ルートです。

## Cowork と Claude Code は有料プランの機能

ひとつだけ先に伝えておきたい大事なこと。**Cowork と Claude Code は、基本的に有料プランの機能** です。

無料プランでもチャット機能は普通に使えますが、この2つを触りたい場合は **Pro 以上のプラン** に加入する必要があります。「デスクトップアプリ自体は無料でダウンロードできるけど、中で使える機能はプランによって変わる」という理解で OK。

### プラン別比較表

2026 年 4 月時点の、主なプランと主要機能をまとめるとこんな感じです。

| プラン | 月額 | チャット | Cowork | Claude Code | 使用量の目安 |
|---|---|:---:|:---:|:---:|---|
| **Free** | $0 | ○ | ✗ | ✗ | 基本量 |

※ 料金は変動するので、最新は [Anthropic 公式の料金ページ](https://claude.com/pricing) で確認してください。年払いにすると月額が少し安くなります。

### どれを選べばいい？

私自身が最初の頃に悩んだポイントも含めて、ざっくり選び方の指針を書いておきます。

- **とりあえず触ってみたい** → **Free** でチャットから試す
- **Cowork や Claude Code を個人で使いたい** → **Pro（月 $17〜）** が一番コスパ良し。個人利用ならここがスタートライン
- **ガッツリ使う / 開発でブン回したい** → **Max（$100〜）** で使用量の上限をあまり気にしなくてよくなる
- **チームで共有したい** → **Team プラン**（1人あたり $20〜）で管理機能もついてくる

私は **Pro から入って、物足りなくなった段階で Max に上げる** 流れが一番ムダがないと思っています。いきなり Max にしてもリターンが活かせないことが多いので、**まずは Pro でしばらく使い倒してみる** のがおすすめです。
