## Claude Code とは

Claude Code（クロード・コード）は、Anthropic が提供する CLI 上で動作する AI コーディングアシスタントです。ターミナルで `claude` と打てば、Claude と会話しながらコードを書いたり、ファイルを直したり、テストを走らせたりできます。

Cursor や GitHub Copilot のようなツールと違って、エディタに縛られないのが最大の特徴。Bash の延長線として AI に作業を投げられる感覚がクセになります。

## インストールと初回起動

Mac / Windows / Linux のどれでも OK。npm が入っていれば、どの環境でも同じコマンドで入ります。

```bash
npm install -g @anthropic-ai/claude-code
cd my-project
claude
```

初回だけ Anthropic アカウントのサインインを案内されます。

## 使いこなしのコツは「普通に話すこと」

Claude Code を触り始めたときに、多くの人が「どんなコマンドを覚えればいいんだろう」と身構えます。でも実は、コマンドを覚える必要はほとんどありません。

> このファイルの関数、バグってるから直して
> テスト全部通るようにして
> README に使い方を追記して

こんな普通の日本語で OK。Claude 側が「どのファイルを読めばいいか」「どのコマンドを叩けばいいか」を判断して、勝手に進めてくれます。

## 最初に伝えたい1つだけのコツ

プロジェクトで初めて `claude` を起動したら、一言でいいので前提を伝えるのがおすすめ。

これだけで、そのあとの会話のテンポが別物になります。

## まとめ

Claude Code は「ターミナル + AI」の気持ちよさが味わえるツール。コマンド表を丸暗記するのではなく、日本語で指示を出すところから始めてみてください。
